Progetto libreria

L'utente dovrà poter:
- Cercare libri per titolo, autore o genere.
- Visualizzare i dettagli di un libro specifico (trama, autore, anno di pubblicazione, valutazione)
- Comprare il libro (quindi utilizzare un database per l'aggiunta e rimozione di quantità dello stesso libro)

Proprietà del mio libro:
- Titolo
- Autore
- Genere
- Trama
- Valutazione

Metodi:
Il metodo SortBooksBy gestisce la logica dell'ordinamento, dove all'utente viene data la possibilità di cercare ed ordinare uno o più libri in base alle proprietà dello stesso (titolo, valutazione ecc...).
ShowAllBooks mostra tutti i libri presenti in libreria.
AddBook permette all'utente di agigungere n libri.
